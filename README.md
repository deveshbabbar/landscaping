# Stoneward Outdoor Living — Demo Marketing Site

A complete, deploy-ready marketing & SEO website for **Stoneward Outdoor Living**, a
fictional award-winning landscaping / outdoor-living **design-build** firm in
Dallas–Fort Worth. Built as a portfolio/demo to show home-improvement,
landscaping, hardscaping, and pool-builder prospects what a high-converting site
looks like.

> **Demo notice:** Stoneward is fictional. All projects, prices, ratings, and
> testimonials are illustrative **sample content**, and every photo is a
> royalty-free Unsplash placeholder. Replace imagery and copy with the real
> business before going live.

## Stack

- **Next.js (App Router)** + **TypeScript**
- **Tailwind CSS** design system (`tailwind.config.ts`)
- `next/image` (optimized Unsplash placeholders) + `next/font` (Sora + Inter)
- `lucide-react` icons
- Zero runtime data services — it is a fully static marketing site

## Edit everything in one place

All editable content — business info, services, projects, reviews, FAQ, financing,
form options, imagery — lives in **`lib/site-data.ts`**. Re-content or re-brand the
whole site from that one file; every page and component reads from it.

### Swap in real photos
Images are Unsplash placeholders generated by the `img()` helper in
`lib/site-data.ts`. To use real project photography:
1. Drop files into `public/images/`.
2. Replace the `img("…")` URLs with leading-slash paths, e.g. `"/images/patio-frisco.jpg"`.
3. The before/after sliders take a `before` and `after` URL each — use genuine
   jobsite "before" shots (the demo "before" frames are muted stock photos).

### Connect the lead form
The **Get a Free Estimate** form (`components/QuoteForm.tsx`) POSTs to
`SITE.formEndpoint`. Set it to your [Formspree](https://formspree.io) form id or a
serverless endpoint in `lib/site-data.ts`. Update `SITE.financingApply` with your
real financing-partner application link, and `SITE.mapEmbed` with a Google Maps
embed for your address.

## Design system

Fresh, high-contrast **modern botanical** — not warm/earthy cream-and-gold:

| Token | Hex | Use |
|---|---|---|
| `mist` | `#F4F6F2` | cool near-white page base |
| `white` | `#FFFFFF` | cards / surfaces |
| `olive` | `#23271E` | brand dark — header, footer, feature blocks, overlays |
| `ink` | `#14160F` | headings & body text on light |
| `stone` | `#5E6157` | secondary text, captions |
| `leaf` | `#7CB342` | accent — CTAs, links, highlights (used sparingly); `leaf-deep #3E6B23` for AA-safe text on light |
| `moss` | `#3E5A2E` | optional deep green for secondary blocks |

Fonts: **Sora** (display/headings) and **Inter** (body), loaded via `next/font`.
Accent green is used only for pops (buttons/links/highlights) and is AA-tuned: the
bright `leaf` is reserved for fills (with dark text) and the darker `leaf-deep`
for green text on light surfaces.

## SEO

- Unique `<title>` / meta description per page; canonical URLs; OG + Twitter cards
- JSON-LD: `GeneralContractor` / `LocalBusiness` + `AggregateRating` + `Service` +
  `areaServed`, plus `WebSite`, `FAQPage`, and `BreadcrumbList`
- `app/sitemap.ts` → `/sitemap.xml`, `app/robots.ts` → `/robots.txt`
- Local keywords woven in (landscaping Dallas, outdoor kitchen builder DFW, paver
  patio Plano, pool builder Frisco, hardscaping Southlake, landscape design Dallas)

Before deploy, set the production domain in `SITE.url` (`lib/site-data.ts`) — it
drives canonicals, sitemap, OG URLs, and all schema `@id`s.

## Accessibility & performance

- Responsive 360–1440px; semantic landmarks; skip-to-content link
- Keyboard-accessible nav, accordion, before/after slider (range input), and the
  multi-step form; visible focus rings
- `prefers-reduced-motion` honored (reveals/animations disabled)
- Optimized `next/image`, lazy maps, minimal client JS

## Getting started

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build
npm start        # serve the production build
```

## Deploy

Deploys to any Next.js host (Vercel, Netlify, a Node server). Remember to:
1. Set `SITE.url` to your production domain.
2. Connect the quote form endpoint and financing/map links.
3. Swap Unsplash placeholders for real project photography.
